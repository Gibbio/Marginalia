# Beta Shared Crates

This directory is the target home for the shared Marginalia Beta engine.

Planned crates include:

- `marginalia-core`
- `marginalia-runtime`
- `marginalia-storage-sqlite`
- `marginalia-provider-fake`
- `marginalia-tts-*`
- `marginalia-stt-*`
- `marginalia-playback-host`
- `marginalia-ffi`
- `marginalia-devtools`

The current Alpha Python implementation remains in place during migration, but
new Beta ownership boundaries should be represented here.
